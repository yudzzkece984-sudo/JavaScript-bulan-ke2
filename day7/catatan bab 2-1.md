apa itu array?
Array adalah sruktur data yang digunakan  untuk menyimpan sekumpulan nilai dalam satu variabel.Nilai-nilai ini bisa diakses menggunakan indeks(dimulai dari 0)

operasi dasar pada array
1.menambahkan elemen
 a.`push()`: menambahkan elemn di akhir array
 b.`unshift()`:menambahkan elemen d awal array
2.menghapus elemen
 a.`pop()`:menghapus elemen terakhir
 b.`shift`:menghapus elem pertama
3.mengakses panjang array
 a.`length`:menunjukkan jumlah elemen dalam array
4.menyisip/menghapus elemen di tengah
 a.`splice(index, jumlah, elemenBaru)`:
 hapus atau tambah
