Studi Kasus: Modul Konversi dan Format
Buat folder `utils/` berisi dua file:
`konversi.js` dengan fungsi `keRupiah(angka)`
 `format.js` dengan fungsi `judul(teks)` untuk mengubah teks menjadi Title Case
Buat file `app.js` yang:
Meng-import fungsi dari dua file tersebut
Mencetak hasil fungsi `keRupiah(1500000)` → "Rp1.500.000"
 Mencetak hasil fungsi `judul("belajar modular javascript")` → "Belajar Modular Javascript"
